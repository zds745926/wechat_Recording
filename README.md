# wechat_Recording
当前版本基于linux操作系统pipewire,可自动创建一个虚拟声卡让微信接入,来录取音频,已经测试暂无bug
可在微信启动程序加上该脚本和微信同时运行,在auto.sh中自行修改音频存放位置

The current version is based on the Linux Pipewire system and can automatically create a virtual sound card for WeChat to connect and record audio. It's been tested and has no bugs so far. You can launch this script along with WeChat, and in auto.sh you can modify where the audio is saved.
